Visualizations: Latitude vs. X


This analysis was done as part of an assignment for George Washington University's Data Analytics Bootcamp. I used CityPy to map 1500 randomly generated latitude-longitude pairs to the nearest valid city. I then passed this list of randomly-generated cities to Open Weather Map's API to pull each city's maximum temperature, cloudiness, humidity, and wind speed at that moment. The data was saved to a Pandas Dataframe and visualized using MatplotLib.

Even though the parameters for pulling cities from Citipy were set to generate locations between -90 and 90 degrees latitude, cities are never generated below -60 latitude. This is because the southernmost tip of South America is at around -55 degrees latitude, and there is only one permanent settlement south of that in Antarctica.
